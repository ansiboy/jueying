# jueying
HTML 页面设计器组件，可以帮助你开发出一个能够集成到系统的设计器。

## 使用到的第三框架

1. React
1. jquery & jquery-ui

## 安装使用

1. 下载 git clone https://github.com/ansiboy/jueying
2. 进入目录 cd jueying
3. 安装依赖的库 npm install
4. 运行演示 grunt dev
